> **580,000+ Americans** experience homelessness on any given night. Evidence consistently shows: housing first works better than treatment-first. But navigating housing programs, applications, and landlord outreach is overwhelming without help.

## The Solution

5 AI agents following the evidence-based HUD/SAMHSA Housing First model. Free forever. GPL-3.0. Grant-funded by OliWoods Foundation.

## Agents

**HousingNavigator** — Match to available housing programs by location and eligibility
**ApplicationAssistant** — Help complete housing applications and gather documentation
**LandlordOutreach** — Generate landlord engagement letters with incentive programs
**StabilityCoach** — Post-housing support and stability planning
**EvictionPrevention** — Detect at-risk households and connect to emergency assistance

## Safety

MANDATORY: 211, domestic violence hotline, veterans crisis line, runaway safeline, 988. Cannot be disabled.

## Data Sources

HUD CoC data, NLIHC affordable housing database, local PHA directories
